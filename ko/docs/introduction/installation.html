Installation

TouchGFX는 다음과 같은 구성 요소를 포함하는 X-CUBE-TOUCHGFX zip 파일의 형태로 배포됩니다.

  • TouchGFX Designer  - Windows 기반의 GUI 빌더를 통해 UI 빌드
  • TouchGFX Generator - STM32CubeMX를 통해 맞춤형 TouchGFX HAL 생성
  • TouchGFX Engine - UI 애플리케이션을 구동하는 TouchGFX C++ 프레임워크

STM32 평가 키트에서의 프로토타이핑​

TouchGFX Designer를 사용해서 STM32 평가 키트에서 프로토타이핑을 시도 해보려면 TouchGFX Designer 설치 섹션을 참조하십시오.  

TouchGFX Designer 설치​

ST.com 공식 웹사이트에서 하드 드라이브로 X-CUBE-TOUCHGFX를 다운로드한 다음 압축을 해제하십시오.

St.com에서 X-CUBE-TOUCHGFX 다운로드

압축이 해제된 폴더를 열면 아래 경로에 TouchGFX .msi 설치 프로그램이 있습니다.

Utilities\PC_Software\TouchGFXDesigner

.msi 파일을 더블 클릭하면 설치 프로그램이 나타납니다. 지침에 따라 설치 프로세스를 완료하십시오.

TouchGFX Designer 설치

Tip
TouchGFX Designer 내에서 직접 보드를 플래싱하려면 다음과 같은 도구가 필요합니다.
  • STM32CubeProgrammer
  • STM32 ST-LINK 유틸리티

STM32CubeProgrammer 설치​

STM32CubeProgrammer download location에서 STM32CubeProgrammer를 다운로드한 후, 다운로드한 .zip 파일의 압축을 해제하고 .exe 설치 프로그램 파일을 실행하십시오. 지침에 따라 설치 프로세스를 완료하십시오.

TouchGFX Designer 설치

Caution
TouchGFX Designer와 Makefiles에서 대상 하드웨어를 플래싱하기 위해 STM32CubeProgrammer를 사용하려면 반드시 기본 설치 위치에 설치해야 합니다.  C:\Program Files\STMicroelectronics\STM32Cube\STM32CubeProgrammer

STM32 ST-LINK 유틸리티 설치​

STM32 ST-LINK 유틸리티 위치에서 STM32 ST-LINK 유틸리티를 다운로드한 후, 다운로드한 .exe 파일을 열어서 지침에 따라 설치 프로세스를 완료하십시오.

STM32 ST-LINK 유틸리티 설치

Caution
TouchGFX Designer와 Makefiles에서 대상 하드웨어를 플래싱하기 위해 STM32 ST-LINK 유틸리티를 사용하려면 반드시 기본 설치 위치에 설치해야 합니다. C:\Program Files (x86)\STMicroelectronics\STM32 ST-LINK Utility

자체 제작 제품 개발​

ST 디스플레이 키트나 자체적인 맞춤형 STM32 기반 플랫폼에서 TouchGFX 애플리케이션을 실행하려는 경우에는 STM32CubeMX에 TouchGFX Generator 설치 섹션을 참조하십시오.

STM32CubeMX에 TouchGFX Generator 설치​

STM32CubeMX를 통해 X-CUBE-TOUCHGFX를 설치하려면 "Help" 아래의 "Manage Embedded Software Packages"를 선택하십시오. (아니면 Alt + U를 누르십시오)).

Help -> Manage embedded software packages

가용 패키지의 최신 목록을 보려면 'Refresh'를 클릭하십시오.

Refresh를 통해 가용 패키지 업데이트

"STMicroelectronics" 탭으로 이동합니다. 화면을 스크롤하여 'X-CUBE-TOUCHGFX'를 찾은 다음 노드를 확장합니다. "TouchGFX Generator"의 확인란을 선택하고 "Install Now"를 클릭합니다. 이제 패키지가 다운로드되고 라이선스 계약이 나타납니다.

TouchGFX Generator 패키지 찾기

라이선스 계약을 읽고 동의한 후 "Finish"를 클릭해서 STM32CubeMX 내부에 TouchGFX Generator를 설치하십시오.

라이선스 계약 수락

X-CUBE-TOUCHGFX 배포판은 아래 경로로 압축이 해제됩니다. 

C:\Users\<user>\STM32Cube\Repository\Packs\STMicroelectronics\X-CUBE-TOUCHGFX\4.13.0
